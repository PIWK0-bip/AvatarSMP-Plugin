# Contributing to AvatarSMP

Thank you for your interest in **AvatarSMP**!

## Project Status & License
Please note that this project is published under an **All Rights Reserved (ARR)** license. 

- **Pull Requests:** External code submissions are closed or evaluated strictly on a case-by-case basis.
- **Bug Reports & Feature Suggestions:** The best way to contribute is by reporting bugs or suggesting new ideas!

## How to Help
1. Go to the [Issues](https://github.com/PIWK0-bip/AvatarSMP-Plugin/issues) tab.
2. Open a **Bug Report** or **Feature Request**.
3. Provide as much detail as possible (server software, versions, stack traces).
