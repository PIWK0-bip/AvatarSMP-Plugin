# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a critical security vulnerability or exploit within **AvatarSMP**, please **do not** open a public issue.

Instead, report vulnerabilities privately:
- **Discord:** Reach out to `PIWK0__bip` directly.
- **GitHub:** Send a private message or security disclosure if enabled.

You will receive a response as soon as possible. Thank you for helping keep the plugin safe!
